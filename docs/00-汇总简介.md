1. **手写简易 Webpack** - 理解核心流程
2. **Loader/Plugin 机制** - 理解生命周期
3. **性能优化案例** - 实际项目经验

## 🛠️ 学习环境准备

### 必装工具
```bash
# Node.js 版本
node >= 16.0.0
npm >= 8.0.0

# 推荐使用 nvm 管理 Node 版本
nvm install 18
nvm use 18
```

### 初始化项目
```bash
mkdir webpack-learning
cd webpack-learning
npm init -y

# 安装 Webpack 核心
npm install webpack webpack-cli webpack-dev-server --save-dev

# 安装常用 Loader
npm install babel-loader @babel/core @babel/preset-env css-loader style-loader sass-loader sass --save-dev

# 安装常用 Plugin
npm install html-webpack-plugin mini-css-extract-plugin terser-webpack-plugin css-minimizer-webpack-plugin --save-dev

# 安装辅助工具
npm install webpack-merge dotenv webpack-bundle-analyzer speed-measure-webpack-plugin --save-dev
```

### 推荐 VS Code 插件
- **Webpack Snippets** - Webpack 代码片段
- **ESLint** - 代码规范
- **Prettier** - 代码格式化
- **Live Server** - 本地服务器

## 📖 学习方法

### 1. 概念 + 实战结合
- 每个知识点先理解概念
- 然后动手实践，不要只看不动手

### 2. 输出倒逼输入
- 尝试自己总结知识点
- 写博客或笔记加深理解

### 3. 源码阅读建议
- 先看简易版实现
- 再看官方源码
- 不必强求全部看懂

### 4. 面试准备
- 准备 2-3 个实际项目案例
- 能描述问题和解决方案
- 了解原理而非只记答案
